This document is not for AI, but as a "get up to speed" document for myself.

This folder contains a Pong implementation as part of the "20 games challenge".

This is my first game.

Gain a basic understanding of Bevy (my game engine/editor of choice) and put together a complete (but small) game!
Don’t worry about making art just yet, you can find a lot of sprites online that you can use. The main focus for now is on the mechanics of the game engine itself.

# Things that you will learn

1. Using your game engine of choice!
2. Creating and destroying objects.
3. Processing player input and moving objects on screen.
4. Detecting and reacting to collisions.
